# 🧶 Muñecos Crochet XO

Página web profesional para la presentación y venta de muñecos artesanales de crochet hechos a mano.

## 📋 Descripción

Sitio web diseñado para mostrar y promocionar muñecos artesanales de tejido crochet de alta calidad, ideales para:
- Cumpleaños y eventos especiales
- Ramos personalizados
- Graduaciones
- Fechas especiales y aniversarios

## ✨ Características

- **Diseño Responsivo**: Compatible con dispositivos móviles, tablets y escritorio usando Bootstrap 5
- **Optimización de Imágenes**: Lazy loading implementado en todas las imágenes
- **Múltiples Secciones**: 
  - Presentación (¿Quién soy?)
  - Misión y Objetivo
  - Catálogo de tamaños (8cm, 10cm, 15cm, 20cm, 30cm, 45cm, 80cm)
  - Galería de trabajos
  - Formulario de contacto
- **Integración con Redes Sociales**: Facebook, Instagram, WhatsApp, TikTok
- **Animaciones Interactivas**: JavaScript para mejorar la experiencia del usuario

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos personalizados y responsive design
- **Bootstrap 5.3.7**: Framework CSS para diseño responsivo
- **JavaScript**: Animaciones e interactividad
- **Fuentes Custom**: Indie Flower, Lobster, Pacifico, Share Tech

## 🚀 Instalación y Uso

### Requisitos Previos
- Servidor web local (XAMPP, WAMP, MAMP, etc.)
- Navegador web moderno

### Instalación

1. Clona o descarga el repositorio en tu carpeta de servidor local:
   ```bash
   cd C:/xampp/htdocs/
   git clone <url-del-repositorio> pagina-muniecos
   ```

2. Asegúrate de que tu servidor local esté corriendo

3. Abre tu navegador y accede a:
   ```
   http://localhost/pagina-muniecos
   ```

## 📱 Breakpoints Responsivos (Bootstrap 5)

El sitio está optimizado para los siguientes breakpoints:

| Tamaño | Breakpoint | Resolución |
|--------|-----------|------------|
| X-Small | None | <576px |
| Small | sm | ≥576px |
| Medium | md | ≥768px |
| Large | lg | ≥992px |
| X-Large | xl | ≥1200px |
| XX-Large | xxl | ≥1400px |
